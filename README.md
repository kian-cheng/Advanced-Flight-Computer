# Advanced-Flight-Computer
A flight computer designed to be feature rich and intuitive to use. Should include features like dual deployment, ignition, active flight stabilisation and data logging. 

## Features

- MCU: STM32H743VIT6
- Barometer: BMP 580
- IMU: ICM-45686
- GPS: ATGM336
- LORA: Ra-02
- USB-C
- Micro SD card reader
- 6 pyro channels
- 6 servo channels
